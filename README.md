# GoogleAdReport

## Usage

Execute the following command line from the project

	ruby -I lib bin/alkhemy [input file] [output file]

	example:

	ruby -I lib bin/alkhemy /Users/marygomez/ad_performance_aug.csv /Users/marygomez/report.html
